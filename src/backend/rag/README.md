                    │
                    ▼
    [ETAPA 3] Indexação  →  Híbrido
                    │
                    ▼
    [ETAPA 4] Retrieval  →  top-k trechos relevantes
                    │

    [ETAPA 5] Geração LLM  →  resposta em texto
